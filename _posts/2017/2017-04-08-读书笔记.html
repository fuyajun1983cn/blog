    简单易行的时间管理法

    • 蕃茄工作法

      • + - 为什么需要番茄工作法
        • 高效处理一个问题
      • 什么是番茄工作法
        • 一种简单易行的，微观时间管理方法
        • 方法步骤
          • 决定任务
            • 任务描述
              • + - 目标
                • 期望达到的一个结果
              • + - 时间
                • 需要几个番茄钟完成
              • + - 阶段
                • 划分每阶段的目的
            • 避免
              • 盲目乐观，不顾既往经验（如定义的时间过短）
              • 忽视目标，没有量化数字(一定要定义完成时间)
              • 过于完美，太精确(允许存在时间上的一些合理误差)
          • + - 设定番茄钟
            • + - 使用便笺式记录
              • + - 任务
                • 如：阅读《番茄工作法》
              • + - 目标
                • 如：总体读完一遍
              • + - 时间
                • 如：五个番茄钟
              • + - 执行
                • 用特殊标记华表一个番茄钟执行完，如X
            • 定时如25分钟
          • + - 执行直到一个番茄钟计时到达
            • 避免任何打扰和中断
          • 短暂休息（3到5分钟）
          • 阶段休息（如每个番茄钟休息15到30分钟）
      • + - 番茄工作法的应用技巧
        • + - 1. 避免干扰和中断
          • + - 干扰源
            • + - 外在干扰
              • 邮件
              • 电话
              • 别人请求
              • 临时事件
            • + - 内在干扰
              • 杂念
              • 身体需要
              • 人脑反应
          • + - 方法一
            • 关闭可能的干扰源
            • 和周围的人打招呼
            • 做好身体上的准备
          • + - 方法二(气绝延迟)
            • 专注做事件
            • 对发生的事情视而不见
          • + - 方法三(接受和记录)
            • 遇到无法避免的中断时就应该中断
            • 简单记录中断的原因
        • + - 2. 调整番茄钟的时间长度
          • 对于初学者，建议一个番茄钟的时间为25分钟
          • 熟练使用后，可根据自身情况适应调整
          • + - 学习一个新方法的原则
            • + - 先僵化
              • 拿过来来，完全按照番茄工作法的步骤和要求机械套用
            • + - 再固化
              • 重复使用，入心入脑，达到熟练掌握
            • + - 后优化
              • 优化掌握的方法，调整为最适合自己的方式
        • + - 3. 处理长事件
          • 任务分解
          • 为每个子任务按照番茄工作法设置目标的时间
      • + - 番茄工作法的扩展应用
        • + - 将一天的时间通过番茄工作法来管理
          • + - 开始工作前
            • 制作当日活动清单
            • 划分优先级
            • 将重要的事情列出来
            • 最好不要超过3件
          • + - 工作结束后
            • + - 总结
              • 花了几个番茄钟
              • 执行情况
              • 出现的问题
        • + - 用于杂事
          • 把杂事分配到1～2个番茄钟上
          • 集中精力在指定番茄钟内处理完杂事
        • + - 何时不用番茄工作法
          • 没有明确目标
          • 难以掌控时间的事情
          • 开心，休闲的事情

      • 个人成功

              • 立即做
            • 重要不紧急
              • 提前做
            • 不重要紧急
              • 授权
            • 不重要不紧急
              • 拒绝
          • 核心技巧
            • 做计划
              • 周计划：把握方向
                • 根据重要性排列出1～3件最重要的事情

                • 尽量每天安排固定的时间来从事
              • 日计划：重在执行
                • 每日检视今天要处理的1～3件核心事情
                • 从第一件最重要的事情开始做
                • 确保基本达成当天的目标

              • 信任
              • 多用责任授权，少用指令授权
              • 需要指导、时间和耐心
            • 学会拒绝

    内向者的优势

    • 认识内向
      • 内向：精力来自于自我
      • 外向：精力来自于外部
      • 内向和外向的三个核心区别
        • 精力的恢复
        • 对刺激的反应
          • 内向的人对外界的刺激反应强烈
        • 深度VS宽度
      • 内向大半基于天生，后天很难改变
      • 内向者的优势
        • 更加专注
        • 更加慎重
        • 更加有计划和深度
      • 内向者的劣势
        • 容易受到别人的忽视
        • 容易产生社交困难
        • 容易受到外界的影响
    • 改进内向
      • 内向者的策略
        • 不要过度勉强自己
        • 认识想认识的人
        • 友好的面孔和适度闲扯
      • 如何让自己在工作中更好

        • 理解外向者更容易获得赞扬

          正确的表达自己

        • 策略一， 想说就说

    如何做计划

    • 为什么要做计划

      • 时间要管理

        事情要管理

        三大作用

        • 清楚自己要做什么

          清楚事情的进展情况

          让自己的时间得以有效利用

      不同计划的作用

      • 固定时间计划

        • 日计划，周计划，月计划

        关键事务计划

        • 项目计划

      如何做日计划和周计划

      • 日计划的关键点

        • 抓住重点事宜，一般不超过3件

          为重点事宜留出弹性时间

          不要安排太满

          一般在前一天晚上完成

        周计划的关键点

        • 抓住重点事宜，一般不超过3件

          抓大放小，非重要事宜不要列入

          不要把时间安排太细

          固定日常事宜不用列入

      计划的检查和调整

      • 一般1~2天检查一次

        重点事宜完成如何

        出现了哪些偏差

        要做什么调整？

      做计划的常见问题

      • 问题1：我记忆很好，不想做计划？

        Ans：好记性不如烂笔头。

        问题2： 我知道做计划，但是我就是懒得做怎么办？

        Ans：坚持做一周，体会甜头。

        问题3：我太忙了，没有时间做计划？

        Ans：越忙越要做计划。

        问题4：我每次做了计划，但总是完成不了？

        Ans：计划目标不要订得太高。

        问题5：我大把时间，没啥事，用不着计划？

        Ans：仔细计划就会有发现。
